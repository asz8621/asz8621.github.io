Hexo 部落格建立指南

發表於 2024-09-10 分類於 Hexo
全面指導您如何建立並配置屬於自己的 Hexo 部落格，從安裝到主題選擇一步到位。

前言

Hexo 是一個快速、簡單且高效的靜態網站生成器，非常適合用來建立個人部落格。本文將指導您從安裝 Hexo 到配置主題的每一步，讓您能夠快速開始使用 Hexo 來記錄生活和工作。

環境安裝

安裝必要工具

在開始之前，您需要安裝以下工具：

  • Node.js（版本 14 或更高）
  • Git
